# Interim Diagnostic Summaries

These CSVs summarize automatic diagnostic scores from completed model runs.

They are included only as interim reproducibility aids. They should not be treated as final safety labels because human annotation and adjudication are still in progress.

Files:

- `gpt41_v04_stress_condition_summary.csv`
- `gpt41_v04_stress_domain_condition_summary.csv`
- `ollama_llama31_8b_v03_condition_summary.csv`
- `ollama_llama31_8b_v03_domain_condition_summary.csv`

Final paper claims should use adjudicated human labels. Automatic scores can be reported as diagnostic or supporting analysis.
